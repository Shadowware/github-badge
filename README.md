## üõ†Ô∏è How to Use github-badge

1. **Create Badges**  
   Once the app is open, you will see options to create different types of badges. You can select badges for skills, projects, or any other relevant information.

2. **Customize Badges**  
   Enter the text you want to appear on your badge. You can choose colors and styles to match your profile.

3. **Copy the Markdown Code**  
   After creating your badge, click on the "Copy" button beside the Markdown code. This code allows you to easily paste the badge into your GitHub profile README.

4. **Add to Your GitHub Profile**  
   Go to your GitHub profile. Click on "Edit profile" and find the section for your README. Paste the code you copied. Save your changes to display your new badge!

## ‚ùì Troubleshooting
